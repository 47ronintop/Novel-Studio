# Novel Studio v0.1.0 Beta

## M17 安装器与发布通道

- 新增 Windows NSIS installer target，同时保留 unpacked directory artifact，便于本地检查和 artifact secret scan。
- 新增本地 beta release channel manifest，并通过结构化 JSON Schema 校验。
- 新增 release notes 生成流程和 release check gate，覆盖 packaging metadata、installer settings、icon asset 与 signing policy。
- beta 发布保持手动流程。release scripts 不执行 push、上传或远程发布。
- 明确签名策略：M17 本地 beta artifact 允许未签名，未来证书签名通过 `CSC_LINK` 与 `CSC_KEY_PASSWORD` 接入。

## 包含的里程碑

- M0-M16 继续包含在当前 beta 线中。
- M17 增加安装器与发布通道准备能力，不新增 telemetry，也不引入真实模型调用。
