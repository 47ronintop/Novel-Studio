# Novel Studio v0.1.0 Beta

## M17 Installer And Release Channel

- Adds a Windows NSIS installer target while keeping the unpacked directory artifact for local inspection and artifact secret scanning.
- Adds a local beta release channel manifest with structured JSON validation.
- Adds release notes generation and a release check gate for packaging metadata, installer settings, icon asset, and signing policy.
- Keeps beta publishing manual. No push, upload, or remote publication is performed by the release scripts.
- Makes signing explicit: M17 local beta artifacts are allowed to be unsigned, and future certificate-based signing is gated by `CSC_LINK` and `CSC_KEY_PASSWORD`.

## Included Milestones

- M0-M16 remain included in this beta line.
- M17 adds installer and release-channel readiness without adding telemetry or real model calls.
