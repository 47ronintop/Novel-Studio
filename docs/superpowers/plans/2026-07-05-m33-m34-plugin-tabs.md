# M33/M34 插件管理与多标签实施计划

**目标：** 插件注册表在设置页可见，章节顶部标签可点击切换。

**架构：** M33 读取 `plugins/plugins.json` 的摘要，不读取 manifest、不执行插件。M34 复用现有项目章节列表和章节选择 bridge，不新增持久化 tab 状态。

**技术栈：** TypeScript strict、React、Vitest、Electron IPC/preload、schema-first JSON。
