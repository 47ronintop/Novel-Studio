# M30 底部面板真实切换与最小内容闭环

版本：1.0 | 状态：M30 已采纳 | 阶段：Post-M18 产品化打磨

## 目标

M30 补齐 M28 中明确禁用的底部面板 tabs。用户点击“工作流运行 / 问题 / 搜索 / 日志”时，应看到对应面板内容，而不是禁用按钮或无反馈入口。

## 范围

M30 包含：

- `DesktopShellState` 增加当前底部面板 tab。
- `WorkspaceShell` 底部 tabs 支持点击切换、`aria-selected` 和可访问标签。
- 底部面板显示四个最小真实内容：
  - 工作流运行：当前 AI 工作流状态、最近运行数量、失败诊断摘要。
  - 问题：当前仍未完成但已明确排期的功能缺口。
  - 搜索：索引条目、当前查询和结果数量。
  - 日志：本地 beta 运行说明、无 telemetry、无真实模型自动调用和安装产物状态。
- Renderer 只更新内存 shell state，不写项目文件。

M30 不包含：

- 完整日志采集系统。
- 完整问题面板和诊断规则引擎。
- 底部搜索结果点击跳转。
- 工作流 DAG 或运行详情可视化。
- 多 tab 编辑器。

## 验收标准

- 底部 tab 不再禁用。
- 点击底部 tab 会调用 `onBottomPanelTabSelect` 并更新 active tab。
- 每个底部 tab 都有独立中文内容。
- `workspace.toggle-bottom-panel` 仍只控制底部面板显隐，不改变 active tab。
- UI 不直接访问文件系统，不新增真实模型调用，不新增 AI 自动写入。
