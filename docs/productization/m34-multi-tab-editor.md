# M34 多标签编辑器

版本：1.0 | 状态：M34 已采纳 | 阶段：Post-M18 产品化打磨

## 目标

M34 将工作区顶部原本禁用的单一章节标签升级为可点击章节标签切换。用户可以从顶部标签切换当前项目已加载的章节，不再看到“后续补齐”的禁用提示。

## 范围

M34 包含：

- 工作区顶部根据项目章节列表渲染章节标签。
- 当前章节标签使用 `aria-selected` 标记。
- 点击其他章节标签复用现有 `project.selectChapter` 流程。
- 没有项目章节时保留当前未命名章节占位。

M34 不包含：

- 独立的持久化打开 tab 集合。
- 关闭 tab、拖拽排序、拆分视图或跨资产 tab。
- 未保存变更冲突处理弹窗。
- 多窗口编辑。

## 验收标准

- 章节标签不再 disabled。
- 点击标签会调用结构化章节选择回调。
- 切换仍由 renderer bridge 调用 Application/preload，不在 UI 访问文件系统。
- 现有章节编辑、保存和版本历史行为不回退。
