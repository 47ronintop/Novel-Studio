# M31 搜索结果点击跳转

版本：1.0 | 状态：M31 已采纳 | 阶段：Post-M18 产品化打磨

## 目标

M31 将 M20 的搜索结果展示推进为可导航闭环。用户点击搜索结果后，应跳到对应章节或故事圣经条目，而不是只看到不可操作的结果列表。

## 范围

M31 包含：

- 搜索结果项升级为可点击按钮。
- 点击章节结果时切回工作区，并选择对应章节。
- 点击 Story Bible 或 memory 结果时切到故事圣经，并选中对应条目。
- 跳转仍走现有 renderer bridge 和 Application/preload 边界。

M31 不包含：

- 搜索结果高亮定位到正文具体行。
- 搜索历史、过滤器或排序 UI。
- 跨文件全文预览展开。
- 文件系统直接访问。

## 验收标准

- 搜索结果按钮有中文可访问名称。
- UI 测试覆盖点击结果回传结构化 result。
- renderer 跳转不新增文件系统访问，不新增真实模型调用。
- 章节跳转复用 `project.selectChapter`；故事圣经跳转复用 `storyBible.selectEntry`。
